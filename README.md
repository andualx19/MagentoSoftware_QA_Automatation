# Magento Software QA Automation

![Java](https://img.shields.io/badge/Java-ED8B00?style=flat-square&logo=java&logoColor=white)
![Selenium](https://img.shields.io/badge/Selenium-43B02A?style=flat-square&logo=selenium&logoColor=white)
![TestNG](https://img.shields.io/badge/TestNG-009999?style=flat-square&logo=testng&logoColor=white)

This project demonstrates QA automation for Magento eCommerce applications using Java and Selenium.  
It includes automated test scripts for key workflows and follows best practices in test automation.

## Technologies
- Java
- Selenium WebDriver
- TestNG / JUnit
- Maven (for project build and test management)

## Project Structure
- `/src`         -> Source code and page objects
- `/tests`       -> Automated test scripts
- `/drivers`     -> Browser drivers (ChromeDriver, GeckoDriver)
- `/reports`     -> Test execution reports

## Running Tests
To run the automated tests:

```bash
mvn test
```
Or run the test classes directly from your IDE.

## Objective
- Learn and apply QA automation techniques in a real-world Magento environment
- Implement automated test scripts using Selenium WebDriver
- Understand and apply Page Object Model (POM) design pattern
- Generate reports for automated test execution
